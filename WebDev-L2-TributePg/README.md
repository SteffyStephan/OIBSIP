# 🌟 Tribute Page – Dr. A. P. J. Abdul Kalam

A beautifully designed and responsive tribute website dedicated to **Dr. A. P. J. Abdul Kalam**, one of India's greatest scientists, visionary leaders, and the 11th President of India. This project was developed as part of the **Oasis Infobyte Web Development & Designing Internship (Level 2 - Task 1)**.

---

## 📖 About the Project

This tribute page highlights the inspiring life, remarkable achievements, and lasting legacy of Dr. A. P. J. Abdul Kalam. The website is designed with a clean and modern interface to provide visitors with an engaging experience while learning about his journey.

---

## ✨ Features

- 🎨 Modern and elegant user interface
- 📱 Fully responsive design
- 👤 About Dr. A. P. J. Abdul Kalam
- 🕒 Timeline of his life journey
- 🖼️ Interactive image gallery
- 🏆 Major achievements section
- 📚 Books written by Dr. Kalam
- 💬 Inspirational quote section
- ❤️ Legacy section
- ✨ Smooth hover effects and animations

---

## 🛠️ Technologies Used

- HTML5

- Google Fonts
