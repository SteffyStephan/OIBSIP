# 📝 TaskFlow

> **Plan Better. Achieve More.**

TaskFlow is a modern and responsive To-Do Web Application built using **HTML, CSS, and JavaScript**. It helps users organize daily tasks, monitor productivity, and stay focused through a clean and intuitive interface.

---

## ✨ Features

- ➕ Add new tasks
- ✏️ Edit existing tasks
- ✅ Mark tasks as completed
- 🗑️ Delete tasks
- 📋 Separate Pending & Completed task lists
- 📊 Live productivity dashboard
- 📈 Task counters & completion percentage
- 📅 Task timestamps
- 💾 Local Storage support
- 📱 Fully responsive design

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Local Storage API

## 🚀 Getting Started

1. Clone the repository.
2. Open the project folder.
3. Run **index.html** in your browser.

No installation or external libraries are required.

---

## 🎯 Project Objective

TaskFlow was created to practice front-end web development by building a complete task management application. It demonstrates DOM manipulation, event handling, Local Storage, responsive design, and dynamic UI updates using pure JavaScript.
