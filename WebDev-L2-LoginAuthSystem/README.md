# 🔐 AuthFlow

A modern and responsive **Login Authentication System** built using **HTML, CSS, and JavaScript**. AuthFlow provides secure client-side authentication with user registration, login validation, password hashing, and protected dashboard access using `localStorage`.

## ✨ Features

- 🔑 User Registration
- 👤 Login with Username or Email
- 🔒 SHA-256 Password Hashing
- 🚫 Duplicate Username & Email Validation
- ✅ Password Validation (Minimum 8 characters & at least 1 number)
- 📂 Protected Dashboard (Accessible only after login)
- 🔐 Session Management using `localStorage`
- 🚪 Secure Logout
- 👁️ Show/Hide Password Toggle
- 📱 Fully Responsive Design
- 🎨 Modern Glassmorphism UI with Animated Background

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
- Web Crypto API (SHA-256)
- Local Storage

## 🚀 How to Run

1. Clone this repository.
2. Open the project in **Visual Studio Code**.
3. Run the project using the **Live Server** extension.
4. Register a new account and log in to access the dashboard.
