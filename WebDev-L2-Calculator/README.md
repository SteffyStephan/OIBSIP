# 🧮 Calculator

A responsive and user-friendly calculator built using **HTML, CSS, and Vanilla JavaScript**. It performs basic arithmetic operations with a clean interface and smooth user experience—without using `eval()`.

## ✨ Features

* 🔢 Perform Addition, Subtraction, Multiplication & Division
* 📱 Responsive design for desktop and mobile devices
* 🧮 Real-time display for inputs and results
* ⌫ Backspace/Delete functionality
* 🗑️ Clear button to reset calculations
* ⚠️ Division-by-zero error handling
* 🔄 Supports continuous calculations (operator chaining)
* 🎯 Built using **Vanilla JavaScript** with `addEventListener`
* 🚫 No `eval()` used — calculations are handled through custom logic

## 🛠️ Technologies Used

* HTML5
* CSS3 (Grid Layout)
* JavaScript (ES6)
